This project contains automated UI tests developed for the DemoQA Text Box feature
as part of the Software Quality Assurance and Testing course.

## Technologies Used
- Java 17
- Selenium WebDriver
- TestNG
- Apache POI (Excel)
- BrowserStack Automate

## Test Description
The automated tests validate the Text Box form using a data-driven approach.
Test data is stored in an Excel file and includes both valid and invalid inputs.
Email field validation is specifically tested.

## Cloud Testing
The test suite was executed on BrowserStack using real browsers:
- Chrome on Windows 11
- Firefox on Windows 11

BrowserStack credentials are securely stored as environment variables.
